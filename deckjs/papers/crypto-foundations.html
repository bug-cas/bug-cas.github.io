Formas de Criptografia

  • Substituição

    Troca-se alguma coisa por outra.

  • Transposição

    Troca-se apenas as coisas de lugar.

  • Ofuscamento

    Muito utilizado para tokens e senhas.

  • Conceito de chave

    Uma maneira de estabelecer um método para cifrar a mensagem e combinar uma chave entre as pontas.

Tipos de Ataques

  • Força Bruta

    Busca descobrir a chave de uma maneira incremental, matematicamente uma hora vai acertar, e ninguém é tão sortudo que acerte de primeira nem tão azarado que acerte apenas na última.

  • Criptoanálise

    Método que procura utilizar formas mais complexas e inteligentes para encontrar mais rapidamente uma chave. (Análise de Frequência, Colas)

Formas de Criptografia

  • Algorítimos de Bloco

    A chave é aplicada em blocos.

  • Algorítimos de Fluxo

    É utilizado um algorítmo com sementes sem um sequência definida. (RC4 faz isso)

Simétrico vs Assimétrico

  • Chave Simétrica

    Possui uma chave única para cifrar e decifrar a mensagem, a alusão é com a uma porta.

  • Chave Assimétrica

    Nesses algorítimos temos duas chaves, uma para cifrar a mensagem, e outra para decifrar, a alusão aqui é o envio de algo com dois cadeados (vou tentar ilustrar).

    Procurar sobre algorítimos de Diffie-Hellman.

Conclusão

  • Utilize os métodos corretos para cada cenário.

  • Não frite seu processamento atoa.
