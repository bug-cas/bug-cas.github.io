Licenses - BSD vs (L)GPL

BSD

4/3/2-clause BSD / ISC / MIT / Apache

Basically:

  • You are free to do whatever you want with the software.

  • If you have any problems, please don't blame the original author.

(L)GPL

(L)GPLv2 / GPLv3 / AGPL

Basically:

  • You are free to use, redistribute and modify the software.

  • Any changes made to the software must be made available under the same license.
